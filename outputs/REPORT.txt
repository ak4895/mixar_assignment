This report presents the results of a comprehensive 3D mesh normalization and
quantization study. Eight mesh models were processed using two normalization
methods (Min-Max and Unit Sphere), quantized to 1024 discrete bins, and
reconstructed to evaluate quality.

KEY FINDINGS:
• Min-Max normalization consistently outperforms Unit Sphere normalization
• All reconstructions achieve excellent quality (MSE < 0.001 threshold)
• 1024 quantization bins provide sufficient precision
• Reconstruction errors are uniformly distributed (no systematic bias)
• Best performance: explosive.obj (MSE: 1.24e-07 with Min-Max)

================================================================================
METHODOLOGY
================================================================================

Normalization Methods:
----------------------------------------
1. MIN-MAX NORMALIZATION
   • Range: [0, 1] per axis independently
   • Formula: normalized = (vertices - v_min) / (v_max - v_min)
   • Inverse: vertices = normalized * (v_max - v_min) + v_min

2. UNIT SPHERE NORMALIZATION
   • Range: Within unit sphere (radius ≤ 1) centered at origin
   • Formula: normalized = (vertices - center) / scale
   • Inverse: vertices = normalized * scale + center

Quantization:
----------------------------------------
• BINS: 1024 (range 0-1023)
• Process: Normalized coordinates → integer bins → dequantized
• Mapping: q = floor(normalized * 1023) for [0,1] range
• Sphere method: Map [-1,1] to [0,1] before quantizing
